# May 2026 Research Summary

The throughline this month is structural: AI tooling is maturing from individual productivity features into enterprise infrastructure that security teams, compliance officers, and IT ops actually care about. Microsoft shipped Agent 365 (governance for agents at scale), Cursor stacked security review agents onto its core product, and Notion pushed its agents into private Slack channels. None of these moves make the tools better for individual users—they make them safe enough for procurement teams to approve.

There's a secondary pattern running underneath: the economics of production AI are breaking. Harrison Chase is naming open-source models as 2026's bet because closed model costs compound too aggressively in agentic workflows. And PMs are realizing that their irreplaceable skill isn't execution—it's judgment about what to build in the first place.

## What Matters

- **PMs building their own leverage is becoming a scalable pattern.** Stripe's Owen Williams vibe-coded Protodash, an internal tool that converts the design system into interactive prototypes in two minutes. It's now the standard for design reviews at Stripe—not a toy, but part of how they work. The unlock wasn't the tool itself; it was treating the PM's own workflow as a product surface worth optimizing. If your team has a mature design system, this is a weekend project, not a quarter-long initiative.

- **The harness, not the model, is where lock-in actually lives.** Switching model providers used to be the switching cost problem. Now it's the orchestration layer. Model providers are deliberately building proprietary agent frameworks and tool-calling conventions to create switching costs that the model layer alone can't provide. If you're evaluating whether to build on OpenAI Assistants API versus LangChain versus an open framework, that tradeoff between convenience and portability is the technical debt question that matters most.

- **Enterprise AI is no longer an experiment—it's now infrastructure with compliance requirements.** Microsoft's Agent 365 shipped with identity, audit trails, and governance controls for agents running across an organization. Cursor added always-on security agents gated to Teams/Enterprise plans. Notion pushed agents into private Slack channels. The pattern is consistent: the blocker for enterprise adoption isn't capability—it's "who is responsible when the agent does something wrong?" Whoever answers that question first wins the enterprise conversation.
