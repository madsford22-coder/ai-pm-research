# March 2026 Research Summary

March's signal is about making products agent-usable, not just human-usable. LangChain shipped CLI-based skills packages that teach agents framework expertise through executable commands rather than documentation—agents invoke `langsmith trace add` instead of reading SDK docs. Vercel added Model Context Protocol support for standardized tool discovery while shipping infrastructure improvements like 15% faster deployments and CDN routing updates that don't require redeployments.

The secondary pattern is operational optimization as differentiation. Companies competed on developer experience details—deployment speed, infrastructure flexibility, model access timing—rather than capability breakthroughs. Windsurf added GPT-5.4 within days of OpenAI's launch with promotional pricing. GitHub announced 60 million code reviews without quality metrics. The shift: infrastructure velocity matters as much as feature velocity.

## What Matters

- **Agent tooling converging around executable interfaces, not documentation.** LangChain's skills packages give agents framework expertise through CLI commands—`langchain agent build --pattern react` scaffolds a ReAct agent without requiring conceptual understanding. This extends Vercel's 69,000+ skills marketplace from February: agent capabilities expand through composable skills and protocol adoption (Model Context Protocol for standardized tool discovery), not better prompts or bigger context windows. For PMs building developer tools, the question shifts from "is our documentation comprehensive?" to "can an agent use our product autonomously?"

- **Infrastructure providers competing on operational details over features.** Vercel's 15% deployment speed improvement compounds for teams deploying 20+ times daily. Their CDN routing decoupling eliminates infrastructure-only deployments—you can iterate caching strategies without triggering builds. Windsurf added GPT-5.4 with promotional pricing (1x credits for self-serve users) immediately after OpenAI's launch, showing model integration speed as differentiation. The pattern: developer experience details (speed, pricing, configuration flexibility) drive competitive advantage when core capabilities commoditize.

- **AI enabling new governance patterns beyond "write faster."** Teresa's Momental case study shows AI detecting strategy conflicts across product artifacts—PRDs targeting enterprises while OKRs prioritize SMB retention. This isn't passive storage (Jira, Notion) but active governance: the system analyzes artifact content, flags contradictions, prompts alignment conversations. The shift from Feb 12's cross-functional coordination: AI surfaces organizational misalignment that humans miss because context lives in different documents owned by different people.
