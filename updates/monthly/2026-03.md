# March 2026 Research Summary

March marked a shift from raw AI capability announcements to infrastructure optimizations and agent-first tooling. LangChain shipped CLI-based skills packages that teach agents framework expertise through executable commands rather than documentation, while Vercel competed on developer experience details—15% faster deployments and CDN routing updates without redeployments. The signal: infrastructure providers and AI platforms optimizing for agent workflows, not just human developers.

The month also revealed automated governance patterns emerging for cross-functional work. Momental uses AI to detect strategy conflicts across product artifacts (PRDs contradicting OKRs, features missing from roadmaps), treating product documentation like code that needs merge conflict detection. For PMs, this suggests AI's organizational impact extends beyond "write faster" to "stay aligned automatically."

## What Matters

- **Agent tooling converging around executable interfaces, not documentation.** LangChain's skills packages let agents invoke `langsmith trace add` to add tracing or `langchain agent build --pattern react` to scaffold agents—no SDK comprehension required. Vercel added Model Context Protocol (MCP) support for standardized agent tool discovery. The architectural question for developer tools: can agents use your product autonomously, or does your API assume human readers who understand concepts? The shift from "document it well" to "make it invokable by agents" affects how you design APIs.

- **Infrastructure velocity as competitive dimension, not capability breakthroughs.** Vercel's 15% deployment speedup and CDN routing decoupled from deployments optimize developer experience details. Windsurf shipped GPT-5.4 immediately after OpenAI's launch with promotional pricing (1x credits for self-serve users), showing model integration speed as differentiation. GitHub announced 60 million Copilot code reviews without quality metrics—scale signals without product insight. The pattern: companies competing on operational details (speed, pricing, model access) rather than capability announcements.

- **AI enabling automated governance for cross-functional alignment.** Momental detects strategy "merge conflicts"—when OKRs prioritize retention but roadmaps load up acquisition features, or when PRDs target enterprises while goals focus on SMB. Teresa Torres' case study shows AI analyzing product artifacts for contradictions, prompting alignment conversations before teams ship conflicting work. The organizational implication: AI shifts from content generation to coordination—surfacing inconsistencies humans miss when artifacts live in different tools and heads.

2. **Vercel** — [CDN Routing Updates Without Redeployment](https://vercel.com/changelog/vercels-cdn-now-supports-updating-routing-rules-without-a-new-deployment) — How decoupling infrastructure configuration from code deployment eliminates unnecessary build cycles when iterating on caching or routing rules.
