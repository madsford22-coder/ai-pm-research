# March 2026 Research Summary

March showed infrastructure providers competing on developer experience details rather than flashy feature announcements. Vercel shipped 15% faster deployments and decoupled CDN routing from code changes, while LangChain launched CLI-based skills packages that teach agents framework expertise through executable commands instead of documentation. The pattern: making tools faster and more agent-friendly through operational improvements.

The month revealed a shift in how agent capabilities expand—not through bigger models or better prompts, but through composable skills and protocol adoption. LangChain's approach (package framework expertise as CLI commands) and Vercel's MCP support signal infrastructure converging around standards for extending what agents can do autonomously.

## What Matters

- **Infrastructure velocity became a competitive dimension.** Vercel's 15% deployment speedup compounds for teams shipping 20+ times daily. Their CDN routing update lets teams iterate on caching and redirects without triggering deployments—separating infrastructure operations from application code. The shift: infrastructure providers competing on removing friction from common workflows, not adding features.

- **Agent tooling converged around executable skills over documentation.** LangChain shipped CLI packages teaching agents LangSmith tracing and LangChain agent construction through commands like `langsmith trace add` and `langchain agent build`. Agents invoke these without understanding framework internals. Teresa's Momental case study showed similar patterns—AI detecting strategy conflicts across PRDs, OKRs, and roadmaps by analyzing artifact content. The architectural shift: agent capabilities expand through skills composition, not comprehension.

- **Standards versus ecosystems tension emerged in agent platforms.** LangChain's proprietary skills format competes with Vercel's Model Context Protocol (MCP) support—both solve agent tool discovery and invocation, but one requires platform lock-in while the other enables cross-framework interoperability. The outcome determines whether agent tooling fragments (custom skill formats per framework) or converges (MCP as universal protocol).
