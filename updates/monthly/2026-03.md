# March 2026 Research Summary

March saw infrastructure platforms and AI tooling providers optimize for a new user: the AI agent. Vercel decoupled CDN routing from deployments and shipped 15% faster builds. LangChain launched CLI-based skills packages that teach agents framework expertise through executable commands rather than documentation. The shift isn't just "make things faster"—it's "make things agent-native."

For product managers, this changes what "good developer experience" means. Your API docs don't matter if agents can't execute your core workflows without human intervention.

## What Matters

- **Agent-first tooling requires executable interfaces, not documentation.** LangChain shipped skills packages with CLI commands like `langsmith trace add` and `langchain agent build --pattern react`—agents invoke these directly instead of reading SDK docs. The architectural insight: documentation teaches humans conceptual understanding; skills give agents executable capabilities. If your developer tool requires reading docs to use it, agents can't adopt it autonomously.

- **Infrastructure configuration is decoupling from code deployment.** Vercel's CDN routing update lets teams modify caching rules, redirects, and traffic routing without triggering deployments. Previously, testing a new caching strategy meant deploying your entire application. Now infrastructure teams iterate independently of code changes. This separation matters for product velocity—if 20% of your deployments are infrastructure-only tweaks, eliminating those from your build pipeline is meaningful time saved.

- **Platform competition shifting to operational details over feature lists.** Vercel's 15% deployment speedup and Windsurf's same-week GPT-5.4 integration with promotional pricing show providers competing on developer experience minutiae. GitHub announced 60 million Copilot code reviews but provided no quality metrics—scale signals without product insight. The pattern: velocity, pricing, and model access matter more than capability breakthroughs when core features have converged.
