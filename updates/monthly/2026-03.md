# March 2026 Research Summary

The story of March is agent platforms growing up—moving from "can we build this?" to "how do we run this at scale?" LangChain rebranded Agent Builder to Fleet, explicitly positioning for enterprise customers managing dozens of agents. GitHub shipped Squad for coordinated multi-agent workflows inside repositories. The shift: individual agent development isn't the constraint anymore. Managing agent fleets, coordinating multi-agent teams, and integrating agents into organizational workflows—that's where the complexity lives now.

Meanwhile, infrastructure providers raced to become the default runtime for autonomous agents. LangChain shipped sandboxes for secure code execution. Vercel launched a plugin letting agents deploy directly to production. Windsurf restructured pricing from credits to predictable quotas. These aren't feature launches—they're positioning moves for who owns the agent execution layer.

## What Matters

- **Agent operations becoming the bottleneck, not agent development.** LangChain rebranded Agent Builder to Fleet, adding centralized agent registries, deployment management, and governance controls for organizations running dozens of agents. GitHub's Squad orchestrates multiple specialized agents (analysis, coding, testing, review) working together on pull requests. The pattern: moving from "how do we build one good agent?" to "how do we manage our agent fleet?"—discovery, versioning, cost control, accountability. If your organization runs more than five agents, you probably can't answer "where are all our agents and what do they cost?" in under five minutes.

- **Infrastructure providers racing to own agent execution environments.** LangChain shipped sandboxes where agents create isolated environments, run arbitrary code, and destroy them programmatically—one line of code. Vercel's plugin closes the deployment loop: agents write code, configure infrastructure, and push to production without human mediation. The architectural shift: agents don't just suggest changes anymore; they execute, test, deploy, and monitor. For PMs building agent products, the question isn't "which LLM should we use?" but "which infrastructure stack has the best agent integrations?"—and that answer increasingly determines build velocity.

- **Pricing models maturing from consumption to predictable quotas.** Windsurf replaced credit-based pricing (different models and reasoning levels consumed different credit amounts) with monthly quotas: Pro gets 500 premium requests, new Max plan gets 2,500 premium requests at 4x price. The Free plan jumped from 2,000 credits to unlimited basic requests. This extends the February pattern (Wispr Flow's team pricing restructure): as AI products mature, pricing converges on predictable SaaS models rather than usage-based systems. Users complained about unpredictability; platforms responded with quotas. The power-user tier at 4x price validates that heavy users exist and will pay for capacity.
