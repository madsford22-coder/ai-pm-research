# June 2026 Research Summary

June was the month the rubber met the road. Everyone shipped AI features in May; June is when the people running production systems admitted what those features actually cost, and started building controls around them. Uber capped Claude Code usage. Figma introduced pay-as-you-go AI credits. Cursor shipped budget management for teams. The conversation shifted from "what can AI do?" to "what can we afford to do with AI?"

Underneath that cost reckoning is a more interesting pattern: the definition of "shipping a product" is changing. Perplexity deprecated its search app to replace it with an OS-level agent. Notion opened its API and explicitly called agents first-class citizens. LangChain shipped self-evaluation loops so agents can correct their own work. The products that are winning aren't the ones with the best models — they're the ones building infrastructure for agents to operate reliably in production.

## What Matters

- **Isolated compute and storage are becoming infrastructure primitives for production agents.** LangChain shipped sandbox environments, AWS Bedrock shipped interactive shells, and Vercel shipped persistent storage drives — all within 48 hours, all solving the same problem. Agents need real OS-level access to do useful work (execute code, write files, run scripts), but that access can't touch your production systems. The converging pattern suggests this layer is getting built out right now, and any PM shipping agent features needs a compute isolation story before they need it.

- **Cost control is now table stakes for AI adoption.** Uber capping Claude Code, Figma introducing pay-as-you-go credits, and Ravi Mehta coining "tokenmaxxing" all point to the same thing: teams are hitting real cost ceilings on AI tooling, and the tools that feel most powerful (long agentic sessions, deep reasoning chains) are exactly what drive the spike. The first companies to hit this problem are solving it with usage caps and visibility — Cursor added context budget reports, Figma added per-team credit controls. This isn't optional anymore; it's a product requirement.

- **The product surface is expanding past the interface.** OpenAI repositioned Codex from a developer tool to a cross-functional productivity platform. Notion opened a Developer Platform and explicitly called agents first-class consumers. LangChain shipped RubricMiddleware so agents can evaluate their own outputs. Perplexity deprecated its Mac app to go all-in on an OS-level agent. The companies winning this month aren't adding AI features — they're becoming infrastructure that agents can extend and operate through. This shifts the PM question from "what AI capability should we add?" to "what role does our product play when the agent is the user?"
