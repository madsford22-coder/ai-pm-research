### Microsoft - Windows 365 for Agents Launch
**Source:** https://blogs.windows.com/windowsexperience/2026/01/22/windows-365-for-agents-the-cloud-pcs-next-chapter/

**tl;dr:** Microsoft launched Windows 365 for Agents, enabling autonomous agents to run on cloud PCs with Windows environments. Shows how infrastructure platforms adapt to support agent deployment.

**What changed:** Microsoft released Windows 365 for Agents, providing cloud PC infrastructure specifically designed for autonomous agent workloads.

**PM Takeaway:**
Agent deployment requires dedicated infrastructure that supports autonomous operation, not just traditional application hosting.

**User problem impacted:**
Organizations need infrastructure to deploy and run autonomous agents at scale without managing physical hardware or traditional VM setups.

**Product surface area:**
Windows 365 - agent-specific cloud PC infrastructure, autonomous agent hosting, and Windows environment provisioning for agents.

**Decision this informs:**
Whether to build agent infrastructure in-house versus using cloud agent platforms, and how to structure agent deployment environments.

**Pattern to note:**
Cloud infrastructure providers launching agent-specific platforms as autonomous agents require different deployment patterns than traditional applications.
