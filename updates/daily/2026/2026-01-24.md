### Microsoft - Work IQ for GitHub Copilot CLI Launch
**Source:** https://www.linkedin.com/posts/satyanadella_so-much-of-dev-work-happens-in-the-context-activity-7420485585376620544-vudJ

**tl;dr:** Microsoft launched Work IQ for GitHub Copilot CLI, enabling developers to bring team and project context into AI coding assistance. Expands AI context beyond individual files.

**What changed:** Microsoft released Work IQ feature for Copilot CLI, providing team and project context to AI coding tools.

**PM Takeaway:**
AI coding tools improve effectiveness by incorporating team and project context, not just individual code files.

**User problem impacted:**
Developers need AI tools that understand team workflows and project context, not just the code they're currently writing.

**Product surface area:**
GitHub Copilot CLI - Work IQ context features, team workflow integration, and project-aware AI assistance.

**Decision this informs:**
How to expand AI tool context beyond individual tasks, what team information to surface in AI tools, and when context sharing improves AI effectiveness.

**Pattern to note:**
AI developer tools evolving from file-level to project-level and team-level context awareness as developers need AI that understands broader workflows.
