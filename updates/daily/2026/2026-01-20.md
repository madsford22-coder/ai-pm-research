# Daily PM Research Update: 2026-01-20

## Summary

Two PM-relevant signals today: Lenny Rachitsky's analysis of ChatGPT apps as distribution channels reveals how third-party developers can reach 800M+ users through the ChatGPT marketplace, and Vercel's launch of an open agent skills ecosystem shows how platforms enable agent capability sharing. Both demonstrate how AI platforms expand through third-party ecosystems.

## Items

### Lenny Rachitsky - ChatGPT Apps as Next Big Distribution Channel
**Author:** Lenny Rachitsky (Former PM at Airbnb; Newsletter author)
**Source:** https://www.lennysnewsletter.com/p/chatgpt-apps-are-about-to-be-the

**tl;dr:** Lenny published analysis on ChatGPT apps as major distribution channel, noting developers can reach 800M+ users in 30 minutes through ChatGPT marketplace.

**What changed:** Published analysis of ChatGPT apps as distribution channel with concrete usage patterns and builder stories.

**PM Takeaway:**
ChatGPT apps provide massive distribution reach (800M+ users) for third-party developers without traditional app store friction.

**PM problem addressed:**
Product builders need low-friction distribution channels that provide immediate access to large user bases without app store gatekeepers.

**How to apply:**
- Evaluate ChatGPT app marketplace as distribution channel for your product
- Design conversational interfaces optimized for ChatGPT user expectations

**Decision this informs:**
Whether to build ChatGPT apps as distribution strategy, how to prioritize conversational interfaces, and what distribution channels maximize user reach.

**Pattern to note:**
AI platforms becoming distribution channels themselves, enabling third-party developers to reach massive user bases through app marketplaces integrated into chat interfaces.

---

### Vercel - Open Agent Skills Ecosystem Launch
**Source:** https://vercel.com/changelog/introducing-skills-the-open-agent-skills-ecosystem

**tl;dr:** Vercel launched open agent skills ecosystem, enabling developers to share and reuse agent capabilities across projects and platforms.

**What changed:** Vercel released open skills ecosystem for agents, allowing capability sharing and reuse.

**PM Takeaway:**
Open agent skills ecosystems enable faster agent development through shared capabilities versus building everything from scratch.

**User problem impacted:**
Developers need reusable agent capabilities instead of rebuilding common skills for every agent project.

**Product surface area:**
Vercel agent platform - skills marketplace, capability sharing, and open ecosystem for agent development.

**Decision this informs:**
Whether to build proprietary agent capabilities versus adopting open ecosystems, how to structure agent skill sharing, and what capabilities to prioritize.

**Pattern to note:**
Platforms creating open ecosystems for agent capabilities, similar to npm for packages or app stores for applications.
